J評価に、
知覚色差 B_color を入れる。

このページは、メインデモへ入れる前のミニ実装です。合成駒内部コスト P に加えて、見た目補助コスト B の中に Lab / ΔE76 ベースの B_color を入れます。

B_color は、RGBやYCbCrの単純距離ではなく、sRGB → XYZ → CIELAB へ変換したあと、合成駒内のLab平均との差 ΔE76 を使います。CIEDE2000は次の改善候補です。

設定

評価式

J = αP + (1-α)B + β(s-τ)^2 + γ(λ-λ0)^2

P = P_Y + λ(P_Cb + P_Cr)
B = 0.7 B_color + 0.3 B_edge
B_color = mean_k mean_i ΔE76(Lab_i,k, meanLab_k)^2 / 10000
